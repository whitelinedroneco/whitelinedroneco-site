# WhiteLine Drone Co — Starter Site

This repository contains a single-file static website (`index.html`) you can deploy with **GitHub Pages** and then **embed in Google Sites**.

## How to Deploy (GitHub Pages)
1. Upload `index.html` to the root of this repository.
2. Go to **Settings → Pages**.
3. Under **Source**, choose **Deploy from a branch**.
4. Branch: `main` • Folder: `/ (root)` → **Save**.
5. Your site will appear at: `https://USERNAME.github.io/REPO/`

## Embed in Google Sites
1. Open your Google Site → **Insert → Embed**.
2. Choose **By URL** and paste your GitHub Pages URL.
3. Resize the embedded frame to fit.

---

- Contact form uses **Formspree**. Replace the `action` attribute with your endpoint.
- Basic `LocalBusiness` schema is included for SEO.
